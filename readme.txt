This plugin provides the super admin user with ability to send customized feedback mail to the customers right from WooCommerce. 

== Description ==

This plugin enables multiple registered users to send customized feedback mail right from WooCommerce to the customers.
Super admin user can set to send bulk feedback mails from single or multiple user-IDs.
Admin can customize the email body or the feedback to be sent to the customer.
Admin can send custom feedback mail to customers based on order status.

== Installation ==

how to install the plugin and get it working.

e.g.

1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
